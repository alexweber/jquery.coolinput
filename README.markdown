# CoolInput

CoolInput is a ridiculously straightforward and lightweight input-hint plugin for jQuery.

* Requires jQuery 1.3+
* Docs: [http://www.alexweber.com.br/jquery/coolinput](http://www.alexweber.com.br/jquery/coolinput)
