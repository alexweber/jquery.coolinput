# CoolInput

CoolInput is a ridiculously straightforward and lightweight input-hint plugin for jQuery.

* Requires jQuery 1.3+
* Docs: [http://code.alexweber.com.br/jquery/coolinput](http://code.alexweber.com.br/jquery/coolinput)
